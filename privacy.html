개인정보처리방침

시행일: 2026년 4월 18일

1. 수집하는 개인정보

QARAH(이하 "서비스")는 이름 추천 기능을 위해 사용자가 직접 업로드하거나 촬영한 얼굴 사진을 처리합니다. 단, 해당 사진은 사용자의 기기(브라우저) 내에서만 분석되며 서버에 전송되거나 저장되지 않습니다.

서비스 이용 시 자동으로 수집될 수 있는 정보:

  • IP 주소, 브라우저 유형, 운영체제 (웹 서버 로그)
  • 쿠키 정보 (Google AdSense 광고 제공 목적)
  • 페이지 방문 기록 (Google Analytics, 사용 시)

2. 사진 데이터 처리

사용자가 업로드하거나 촬영한 사진은 다음과 같이 처리됩니다:

  • 사진은 사용자의 브라우저 내 JavaScript 엔진에서 직접 분석됩니다
  • 사진 데이터는 외부 서버로 전송되지 않습니다
  • 분석 완료 후 사진 데이터는 브라우저 메모리에서 즉시 삭제됩니다
  • 서비스 제공자는 사용자의 사진에 접근할 수 없습니다

3. 광고 서비스

본 서비스는 Google AdSense를 통해 광고를 제공합니다. Google은 쿠키를 사용하여 이전 웹사이트 방문 기록을 기반으로 관련 광고를 게재할 수 있습니다. Google의 광고 쿠키 사용에 대한 자세한 내용은 Google 광고 정책을 참조하세요.

4. 쿠키 정책

본 서비스는 광고 게재 및 사이트 분석을 위해 쿠키를 사용할 수 있습니다. 브라우저 설정을 통해 쿠키 수집을 거부할 수 있으나, 일부 서비스 이용이 제한될 수 있습니다.

5. 개인정보의 보유 및 파기

사진 데이터: 분석 즉시 삭제 (서버 저장 없음)

서버 로그: 최대 30일 보관 후 자동 삭제

6. 이용자의 권리

사진 데이터가 서버에 저장되지 않으므로, 별도의 삭제 요청이 필요하지 않습니다. 서비스 이용에 관한 문의는 문의 페이지를 이용해 주세요.

7. 아동의 개인정보

본 서비스는 만 14세 이상의 이용자를 대상으로 합니다. 만 14세 미만의 아동은 법정 대리인의 동의 하에 서비스를 이용해야 합니다.

8. 변경사항

본 개인정보처리방침은 관련 법률 변경 또는 서비스 변경에 따라 수정될 수 있습니다. 변경 시 본 페이지를 통해 공지합니다.
